# FitHUb Website — Assets Upload Guide

## 📁 Folder Structure

Upload your photos and videos to the correct folders below:

```
assets/
├── logo/
│   └── fithub-logo.png          ← FitHUb logo (PNG with transparent background)
│
├── founders/
│   ├── mitesh-patel.jpg          ← Mitesh Patel's coach photo
│   └── trupti-patel.jpg          ← Trupti Patel's coach photo
│
├── testimonials/
│   ├── trupti-before.jpg         ← Trupti before transformation
│   ├── trupti-after.jpg          ← Trupti after transformation
│   ├── trupti-avatar.jpg         ← Trupti profile circle photo
│   ├── mitesh-before.jpg         ← Mitesh before transformation
│   ├── mitesh-after.jpg          ← Mitesh after transformation
│   ├── mitesh-avatar.jpg         ← Mitesh profile circle photo
│   ├── pratik-before.jpg         ← Pratik before transformation
│   ├── pratik-after.jpg          ← Pratik after transformation
│   ├── pratik-avatar.jpg         ← Pratik profile photo
│   ├── aayushi-before.jpg        ← Aayushi before transformation
│   ├── aayushi-after.jpg         ← Aayushi after transformation
│   └── aayushi-avatar.jpg        ← Aayushi profile photo
│
├── services/
│   ├── wellness-eval.jpg         ← Wellness evaluation service image
│   ├── meal-plans.jpg            ← Meal planning image
│   ├── workouts.jpg              ← Workout sessions image
│   └── coaching.jpg              ← 1:1 coaching image
│
├── backgrounds/
│   ├── hero-bg.jpg               ← Hero section main image (wide/landscape)
│   └── center-interior.jpg       ← FitHUb center interior photo
│
└── videos/
    ├── training-video.mp4        ← Training session video (optional)
    └── testimonial-video.mp4     ← Client testimonial video (optional)
```

## Image Recommendations
- Photos: JPG or PNG, minimum 800x800px
- Before/After: Portrait orientation preferred  
- Founder photos: Professional quality, 600x800px minimum
- Hero background: Wide/landscape, 1200x800px minimum
- Videos: MP4 format, under 20MB each

## Where Images Appear
| File | Location on Website |
|------|---------------------|
| logo/fithub-logo.png | Top navigation bar |
| founders/mitesh-patel.jpg | Coaches section - Mitesh card |
| founders/trupti-patel.jpg | Coaches section - Trupti card |
| backgrounds/hero-bg.jpg | Hero section (top of homepage) |
| backgrounds/center-interior.jpg | About section |
| testimonials/*-before/after.jpg | Transformation stories section |

Note: Until photos are uploaded, the website shows placeholder icons automatically.
